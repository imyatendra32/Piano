🎹 Virtual Piano Web Application

📌 Project Overview

The Virtual Piano is a web-based interactive application that allows users to play piano notes directly in their browser. Users can either click on the piano keys or use their keyboard to produce sound.

---

🚀 Features

- 🎵 Play piano sounds using mouse clicks
- ⌨️ Keyboard support for playing notes
- 🎨 Clean and responsive user interface
- ⚡ Real-time sound interaction

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

📂 How to Run the Project

1. Clone the repository:
   git clone https://github.com/imyatendra32/Piano.git

2. Open the project folder

3. Run the "index.html" file in your browser

---

🎯 Purpose of the Project

This project was developed to enhance front-end development skills and understand JavaScript event handling, DOM manipulation, and audio integration.
